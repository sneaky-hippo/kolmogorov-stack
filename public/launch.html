kolm: your own AI, in a single file.

I built kolm because I wanted to ship AI features without sending my customers' data to OpenAI on every request. The shape that worked is a compiler: take a task description, the user's data, and a frontier model API key — emit one signed file that runs offline anywhere. Same artifact runs on my laptop, my phone, my server, my edge box. The compiler is the moat; the runtime is yours.

02guarantees

What every .kolm guarantees.

Not a leaderboard claim. A set of properties enforced by src/verifier.js at compile time and by src/benchmark.js at run time. If any property fails, the build doesn't sign.

compile-time gatequality_score ≥ 0.85 AND positive_pass_rate ≥ 0.85 against the embedded eval set. Fail → no artifact.
verified-inference mathk-sample frontier + sound verifier → accuracy = 1 − (1 − p·v)k. Monotone in k by construction.
local executionvm.Script sandbox; latency in microseconds via process.hrtime.bigint. No network call to run.
egress monitorbench harness patches fetch / http / https / net / tls / dns. Any phone-home is recorded.
integrityHMAC-SHA256 over manifest.json. signature_valid: true or the artifact is rejected.
receipt chaintask → seeds → recipes → evals → package, each step seals the previous.

The artifact is the benchmark. Every .kolm ships with the eval set used to gate it. kolm bench file.kolm reruns those evals offline and emits a signed JSON report. That is the unit of proof. → benchmark spec & harness source

03reproduce

Run the harness yourself.

No magic numbers. The harness is one file. You compile an artifact, you run the bench, you get JSON. The same artifact on different machines should produce the same accuracy and the same signature hash.

$ git clone https://github.com/sneaky-hippo/kolmogorov-stack
$ cd kolmogorov-stack
$ npm install
$ export ANTHROPIC_API_KEY=sk-ant-…

04what it isn't

Scope bar.

not a quantized 70BWe don't claim to run frontier weights on a phone. We compile a small model whose behavior on your task matches frontier — because it was distilled on your task.
not a cloud API wrapperYou pay your model API bill once, at compile. Run-time is local at zero marginal cost.
not new trainingBases are open weights (Qwen2.5, Llama-3, Phi-3). We add a LoRA delta and a recipe pack. The teacher is the frontier you already use.
not a magic answerIf your gold examples are bad, the K-score gate catches it. The compiler refuses to sign artifacts that fail the floor.
