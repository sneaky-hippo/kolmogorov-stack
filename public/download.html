One CLI. Three native wrappers.

The kolm CLI is the canonical entry point. The macOS and Windows apps wrap the same binary and the same runtime; the Linux install is a curl-pipe of the same script the CLI uses on every other platform.

CLI · Node 20+

The fastest path. Installs the kolm binary globally; works on macOS, Windows, and Linux without admin.

npm i -g @kolm/cli

verify with kolm --version · then kolm login

macOS app · Apple silicon + Intel

A native app that wraps the CLI, the runtime, and the capture inbox. Signed with a Developer ID; first launch asks for the receipt secret only.

npm i -g kolm

Linux · npm one-liner

Installs the CLI and the runtime under ~/.kolm. Adds the binary to your shell rc on first run; no root, no systemd unit by default.

npm i -g github:sneaky-hippo/kolmogorov-stack
