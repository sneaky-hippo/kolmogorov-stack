01The contract

What you give kolm.
What kolm gives back.

A clean exchange. Everything on the right is reproducible from everything on the left. Nothing leaves your machine that you didn't ship in.

You provide

Task - examples - evals - key.

  • taskOne sentence. The behavior the artifact has to match.
  • examples10-50,000 rows. JSONL, CSV, a folder of files, or a corpus namespace.
  • evalsHeld-out set. Synthesized if absent. The K-score is measured against this.
  • api keyBYO. Anthropic, OpenAI, Google, Mistral, Together, Fireworks, your own gateway.
  • basePick one from the open list, or let kolm choose by size and target.
kolm returns

One signed .kolm. Nothing else.

  • artifactSingle zip. Model, LoRA, recipes, recall index, tests, manifest, signature.
  • k-scoreOne number. Below the gate, the artifact does not ship.
  • receiptHMAC chain. Bound in the artifact receipt. Tamper-detectable.
  • runtimeSame file runs through the CLI, the MCP server, and native iOS / Android adapters: one signature, every surface.
  • costFrontier bill is yours. Local runtime is unlimited on every paid tier.

03Developer experience

One command in the terminal.
One step in CI.

No GPU on your laptop. No Docker. No multi-day MLOps. The compile pipeline lives in the cloud; the artifact lives wherever you put it.

One command

kolm compile handles compile, cache, sign, and serve. CLI distributes via curl, npm, brew, pip.

Reproducible

Same task + same examples + same base = byte-identical artifact. CI gates on K-score, not vibe.

Inspectable

kolm inspect a.kolm shows the manifest, recipes, K-score, and signature metadata. kolm diff a.kolm b.kolm compares two builds field by field.

BYO key

Anthropic, OpenAI, Google, Mistral, Together, Fireworks, AWS Bedrock, Azure, your own gateway.

CI native

GitHub Action, GitLab template, Jenkins step. Fail the build if K drops below your floor.

Customer-hosted bridge

A customer-hosted compile bridge is the Enterprise design path for private networks. The public CLI does not ship an --air-gap mode today.
