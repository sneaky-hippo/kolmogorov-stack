·FAQ

The questions buyers actually ask.

If yours isn't here, mail hello@kolm.ai.

Who pays for the model API calls?

You do. kolm uses your model API key during compile. We never resell tokens, never mark up frontier inference, never bill you for usage we didn't run on our infrastructure.

What counts as a "compile"?

One full pass: synthesize the verifier, k-sample the teacher, train the LoRA, build recipes, package, sign. kolm rebuild with the same inputs is byte-identical and free.

What if K-score is below the gate?

The artifact does not ship. You see the breakdown by component, can adjust seed examples or evals, and recompile. We don't charge for sub-gate compiles on Pro and Enterprise.

How big is a typical .kolm file?

20–500 MB. The base is a 1B–7B open-weight INT4 quant. The LoRA is 20–100 MB. The recall index scales with corpus size; we shard above 4 GB.

Can I run it offline?

Yes. The artifact is fully self-contained. kolm serve --mcp file.kolm requires no network at runtime. Compile is online; run is offline.

Which frontier models can I use as the teacher?

Anthropic, OpenAI, Google, Mistral, Together, Fireworks, AWS Bedrock, Azure OpenAI, your own gateway. Any HTTP completion endpoint that follows the OpenAI or Messages contract.

Which open-weight bases can I compile against?

Qwen 2.5, Qwen 3.x, Llama 3.x, Phi-3, Hermes 3, Gemma 2, at INT4, sized 1B to 14B. Enterprise tier supports private base models you provision.

Is there a free trial of Pro?

Yes. The first 14 days of Pro are free; cancel any time. Artifacts you compiled during trial stay yours and remain runnable forever.

Can I export everything if I leave?

Yes. The artifact is a signed zip; you already own it. The recall index exports as Parquet. Receipts export as JSONL. Nothing is locked behind our cloud.

Do you offer non-profit or academic pricing?

Yes. Pro is free for verified non-profits and accredited academic researchers. Mail us with a .edu or .org address and we'll set it up.
