Boilerplate

Drop-in copy for the lede or about-section. Cleared for use in any publication.

kolm is the AI compiler. It takes a task description, your data, and a frontier model API key, and produces a portable, signed .kolm artifact (a sub-3GB ZIP) that runs fully offline on phone, laptop, browser, server, or edge. Every output is cryptographically receiptable; every release is gated on a held-out K-score. Built by Kolmogorov; v0.1 live at kolm.ai. - 65 words / company-approved

Every .kolm ships with the eval set it was gated on. kolm bench file.kolm reruns those evals offline, monitors network egress (fetch / http / https / net / tls / dns are patched), and emits a JSON report conforming to spec kolm-benchmark-1. The artifact is its own benchmark; the harness is one file in src/benchmark.js.
