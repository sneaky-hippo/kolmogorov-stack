Developer path

From npm i to wired-up IDE in four commands

kolm is one binary, three primitives, and a receipt chain. Install, capture, distill, integrate. Each step prints what it did and signs the result.

01 . install
02 . capture
03 . distill
04 . IDE

Install the CLI

Pick your package manager. The CLI ships as a single statically linked binary plus a thin Node wrapper; no daemon required.

$ npm i -g github:sneaky-hippo/kolmogorov-stack $ kolm login --key ks_xxxxxxxx ok. config written to ~/.kolm/config.json $ kolm doctor CLI version 7.x ready.
$ npm i -g github:sneaky-hippo/kolmogorov-stack $ kolm login --key ks_xxxxxxxx ok. config written to ~/.kolm/config.json $ kolm doctor ready.
PS> npm i -g github:sneaky-hippo/kolmogorov-stack PS> kolm login --key ks_xxxxxxxx ok. PS> kolm doctor ready.
Switch path

kolm capture your first prompt

Captures turn any LLM call into a signed pair you can later replay, distill, or audit. Run this in your existing app or in a one-off REPL.

$ kolm capture --provider openai --tag my-app listening on http://127.0.0.1:11434 (drop-in for any OpenAI client). capturing prompt + response pairs into ~/.kolm/captures/my-app.jsonl press Ctrl-C to stop. 42 pairs captured so far. $ kolm tail captures --namespace my-app --limit 3 3 most recent capture pairs.

why captures

You need at least 100 pairs to distill a useful recipe and 1000 pairs to compile a specialist. The capture stream is local-first; nothing leaves your laptop until you opt in.

kolm distill your first artifact

One command takes your captured pairs to a signed .kolm artifact you can run anywhere. The preview flag prints the plan without writing files.

$ kolm distill --from-captures --namespace my-app --preview plan: recipe class = "extraction" (you have 380 pairs). teacher: deepseek-v4-flash. base: qwen3-vl-7b. estimated compile time: 3m20s. estimated cost: $0.42. $ kolm distill --from-captures --namespace my-app artifact: my-app.kolm. K-score: 0.91. receipt signed. $ kolm verify my-app.kolm signature OK. Rekor entry: 0x7e4f...

what you get

A single .kolm file you can drop into any container, edge function, or device. It already includes the receipt chain, the spec, and the eval set.

Wire it to your IDE

kolm ships first-party MCP installers for Cursor, Claude Code, and Continue. Pick yours and run one command. No server config to maintain.

Cursor

Adds a kolm tool to the MCP palette. Receipts surface in the right pane.
Claude Code

Drop-in slash command. Stream completions through your local artifact.
Continue

VSCode extension. Use your artifact as a chat or autocomplete provider.
VSCode

Generic VSCode extension. Works with any artifact path.
JetBrains

Single plugin for IntelliJ, GoLand, PyCharm, RustRover, and friends.
Zed

Native Zed assistant provider. Configuration is one settings.json key.
