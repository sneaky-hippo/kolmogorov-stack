あなた自身のAIをコンパイルする。

kolm は、一文で記述されたタスクを .kolm アーティファクトに変換するコンパイラです。アーティファクトはあなたのデータで学習され、あなたのハードウェア上で実行され、各呼び出しに署名付きの领収を残します。呼び出し毎の見積も、ベンダーロックインも、学習データの流出もありません。モデルは永続的にあなたのものです。

APIキーを取得 → 60秒クイックスタート 主権AI

4つの動詞、1つの契約。

モデルのライフサイクル全体は、CLI の 4つの動詞に収まります。タスクを記述し、アーティファクトをコンパイルし、ローカルで実行し、各応答を検証する。隠されたサーバーも隠されたテレメトリも隠された料金体系も存在しません。見えているものが、実行されるすべてです。

kolm compile

コンパイルする。

一文の記述とあなたのサンプルデータから、コンパイラは決定論的な .kolm アーティファクトを生成します。署名付き领収チェーン付き、ブロックどごとに再現可能。

kolm run

実行する。

アーティファクトはあなたの GPU、Mac、エッジデバイス、またはサーバーラックで動作します。外部・コールバックは一切ありません。

kolm verify

検証する。

各呼び出しは署名付き领収を残します。ドリフトは検知され、アーティファクトの世代交替は追跡可能、監査人はオフラインで検証できます。

日本の規制次業務に向けて。

kolm は最初から規制対象ワークロードを想定して設計されています。個人情報保護法に構わだ浏通、金融機関の FISC 安全対策基準、医療機関における 3 省2 ガイドライン、自治体におけるローカル LGWAN 構致可能性まで、アーティファクトを 自社境界内に留めたまま運用できます。契約書面は初回依頼から 48 時間以内に創業者審衖されます。

データ保護

越境移転なし。

学習データ、モデル重み、呼び出し履歴はあなたが配置した場所に留まります。日本国内リージョンでのコンパイルを選択すれば、海外転送は発生しません。

監査

各呼び出しを検証可能に。

医療・金融・酉間ドローン・公共ソリューションで求められるアウトプットの逆負けは、领収チェーンにより原因追跡が因数分解できます。
