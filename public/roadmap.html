public commitments

What ships today. What is dated.

The roadmap above traces the internal engineering plan. This section is the original public roadmap — what an outside reader can hold us to with calendar dates.

now

Shipped, in production

today · kolm.ai

Everything below is live on kolm.ai, runs in the public CLI, or sits inside any .kolm you compile this minute. Source on GitHub.

shipped

Apache-2.0 runtime wave 101

The kolm CLI and the runtime that executes a .kolm artifact ship under Apache-2.0. Source on github.com/sneaky-hippo/kolmogorov-stack. Run it on your laptop without us.

shipped

.kolm spec, RS-1 (+ §7.7 cross-vendor) wave 158

RS-1 is the wire format for a compiled artifact: manifest, spec, policy, optional sidecar, verifier, receipt. Section 7.7 covers cross-vendor distillation provenance. Read RS-1.

shipped

Stripe billing, live

Self-serve checkout on Free, Starter, Pro, Teams, Business, Enterprise. POST /v1/account/change-plan returns a Stripe payment link bound to your tenant.

shipped

HMAC-SHA-256 signing (Ed25519 next)

Every .kolm carries a single HMAC over its manifest + layer hashes. Every /v1/run emits a receipt that the issuer or any holder of the shared tenant secret can re-verify offline. Q+8 (Ed25519) replaces this with public-key crypto.

shipped

Cross-vendor distillation wave 158

5 teacher vendors (Anthropic, OpenAI, Google, xAI, local) × 13 student bases (SmolLM2, Llama-3.2, Phi-3.5, Gemma-2, Mistral, Qwen2.5, DeepSeek-V3, GLM-4, Yi-1.5). Full license matrix. See /training.

shipped

PHI / PII redactor wave 157

21 HIPAA-mapped redaction classes (18 Safe Harbor + NPI / DEA / state Medicaid). Receipt chain extends to redaction_map_hash + teacher_call_log_hash + reinjection_log_hash. Raw PHI never leaves the tenant boundary.

shipped

Native compile: Rust + C + WASM wave 148–150

Rule-class recipes compile to deterministic Rust crates, C99 source, and wasm32-wasi modules. Verifier rebuilds from pinned toolchain and confirms binary_hash matches (P+4, wave 156).

shipped

seeds.jsonl train / holdout gate wave 147

Deterministic 80/20 split by row hash. Recipe trains on train pairs only; K-score runs against unseen holdout. Verifier rejects any artifact where train_hash ∩ holdout_hash ≠ ∅.

shipped

Registry catalog, public

A public read-only catalog of recipes, base-model fingerprints, and RS-1 entries at /v1/registry/public. Bulk export at /v1/registry/export.

shipped

MCP harness wire-up

kolm install claude-code, cursor, continue, cline. Writes mcpServers.kolm into the right config; stdio default, optional HTTP JSON-RPC on 127.0.0.1:8765.

next

In flight, v7.X target

v7.X · target 2026-Q2/Q3

Work that is actively underway. Code in the repo, work tracked in the changelog. Targets slip publicly.

v7.X target

K-score T axis + A/T reporting wave 160

Teacher accuracy on the same holdout, reported as A/T. Makes the cost/quality tradeoff legible: a 0.9 A/T artifact running at 5% of teacher cost is the right ship for many workloads. Q+3c per Wave 144 Doc 7 §4.7.

v7.X target

SLSA Level 3 supply chain

Hermetic builds with provenance attestations published per release. CycloneDX SBOM emitted alongside the artifact. Closes the verifiable-build-of-the-runtime gap.

v7.X target

SOC 2 Type I

Auditor engaged for a point-in-time SOC 2 Type I. Scope: production environment, change management, access control, monitoring. Report shareable under NDA.

v7.X target

Sigstore artifact anchoring Q+9

Cosign-signed release artifacts and Rekor transparency-log entries for every .kolm binary we publish. Independent public-key verification, no shared secret required.

v7.X target

Ed25519 receipts Q+8

Per-call receipts signed with a tenant Ed25519 key. Anyone holding the public key can verify a run happened, without our service.
