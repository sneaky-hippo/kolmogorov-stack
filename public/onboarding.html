Install the CLI

One binary. Pick the install path that matches your shell.

npm i -g github:sneaky-hippo/kolmogorov-stack kolm --version # kolm 0.1.0 kolm login # paste your ks_... key when prompted # OK saved to ~/.kolm/config.json

Compile your first .kolm

Pick a starter. We will spec the recipe, fit a small specialist, gate on K-score, and sign the artifact. Average: 90 seconds.

kolm new phi-redactor --from healthcare/phi kolm compile spec.json --base qwen2.5-3b --preset lora-fast --k-gate 0.92 # spec.json -> cloud compile # fit -> evaluate -> gate -> sign # OK ~/.kolm/artifacts/phi-redactor-9a3f.kolm (K=0.982)

Run + verify

The .kolm runs locally. Every call emits an HMAC receipt over (artifact CID, input, output, K-score). You can verify months from now without our servers.

kolm run ~/.kolm/artifacts/phi-redactor-9a3f.kolm \ "Patient John Doe, DOB 03/14/1972, SSN 555-12-3456" # output: Patient [NAME_1], DOB [DATE_1], SSN [SSN_1] kolm inspect ~/.kolm/artifacts/phi-redactor-9a3f.kolm # OK signature OK, K=0.982, receipt chain intact
