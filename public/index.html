3 · Reproducibility you can put on a court exhibit

Cryptographic receipts. RS-1 open spec.

Every run returns an HMAC-SHA256 receipt over canonical JSON: (source_hash, input_hash, output_hash, runtime_version, time). Verifiable offline. Forever. Without us. Stochastic models cannot sign their work — even at temperature 0, retries diverge at the token level. Recipes can. 4 · Composability of programs, not prose

An LLM output is a string. A recipe is a function. Functions compose; strings don't. You can search recipes by semantic similarity, dispatch by routing, chain them, version them, diff them, lint them, ship them through CI. When you should not use Recipe

We're not the answer to every problem. Honest infra means honest scope.

  • Open-ended generation. "Write a marketing email" — use Claude / GPT directly. (We pass through.)
  • Long-context reasoning. "Read this 80-page PDF and extract clauses" — long-context model wins.
  • Multi-step agent loops. Browse, plan, act — Claude Code, OpenAI Assistants, real agents.
  • Domain knowledge with no examples. "Is this molecule toxic?" with no labeled data — you need a real classifier.
  • Recall over a corpus. "Find prior conversations with this customer" — that's Memory, not Skills.
  • Tasks where the right answer changes with world state. "Is this stock a buy?" — fresh inference, not a fixed recipe.

The infrastructure thesis · why this rewires the internet

Frontier-grade behavior, on every device that has ever existed.

Roughly 80% of structured AI calls — classify, route, extract, validate, score — are a trillion-parameter model being asked to do what a 400-byte function could do in 35 µs. Recipe compiles those calls, once, against a verifier the frontier model itself helped build. The output is a portable, signed, deterministic function. Now the structured behavior of the largest models on earth runs on the smallest devices, forever, for free.

The registry is the model. The registry is downloaded, not called. The internet stops shipping AI's structured-task traffic the same way it stopped shipping every cacheable byte two decades ago — and the moment you internalize that, the whole stack changes shape.
