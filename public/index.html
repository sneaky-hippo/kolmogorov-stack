02Compile

One command turns a task into a signed artifact.

Point kolm at a task description and a directory of examples. It synthesizes a deterministic verifier, k-samples a frontier model under the verifier, distills the survivors into a personal LoRA, observes the deterministic-token subset and packs everything as a single signed zip.

~/work/support-bot — kolm compile

04K-score

One number you can stare at. Five you can defend.

The K-score is the harmonic average of size, accuracy, latency, cost and coverage, normalized against the frontier baseline you measured at compile time. Below the gate it does not ship. Above the gate it gets a manifest entry.

K-score
0.94

Harmonic average. 1.0 means the artifact matches the frontier on accuracy at zero marginal cost. Anything below 0.70 is rejected by default.

05Compounding

Every compile makes the next one smaller, faster, cheaper.

kolm watches the live MCP traffic. When a call matches a compiled artifact, the cache wins. When the inputs drift, the residual goes back to the frontier and the next compile picks it up.

01

First call

Frontier model serves it. Receipt is recorded. Inputs and outputs land in the corpus.

02

Repeat call

Cache hit. Local artifact answers in ~80 ms. The receipt chains to the original.

03

Drift detected

Verifier flags an input outside coverage. Frontier resolves it. Coverage gap is logged.

04

Recompile

Next nightly run absorbs the gap. K-score ticks up. The cache strictly grows.

06Install

Five surfaces. One artifact. Same receipt.

A .kolm artifact compiled on your laptop runs unchanged on a phone, in a browser, in a backend, or in a JetBrains plugin.

# macOS, Linux, Windows (WSL)
curl -sSL kolm.ai/install.sh | sh
kolm compile "triage support tickets" --data ./tickets
kolm serve --mcp support-triage.kolm
npm i @kolmogorov/kolm

// Node, Bun, Deno, browser via /sdk.js
import { compile, run } from '@kolmogorov/kolm';
const artifact = await compile({ task, examples });
const out = await run(artifact, input);
pip install kolm

from kolm import compile, run
artifact = compile(task="triage tickets", examples="./tickets")
out = run(artifact, input)

import Kolm
