What's in the math, what isn't.

In: per-call frontier cost (input + output), ~8% egress + variance buffer, kolm plan, teacher cost ((k · rows · in_tokens) at same rate).

Out: engineer time, GPU rental during fine-tune (Pro handles this on the bridge), one-time recall index population, and the cost of not compiling (cloud variance, prompt regressions, rate-limit incidents).

Assumption: kolm fits tasks 80%+ deterministic on labeled examples. For free-form generation, see /k-score or GitHub Discussions.
