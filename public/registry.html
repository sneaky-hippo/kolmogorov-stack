Programmatic API

Same registry, no browser. CID-verified responses, JSON only. Auth is optional for public artifacts.

search by task

curl -s https://kolm.ai/v1/registry/search?task=phi-redact \
  -H "Accept: application/json"

get one by CID

curl -s https://kolm.ai/v1/registry/get/cidv1:sha256:7a2c1f8b9d4e \
  -o phi-redactor.kolm

list page (paginated)

wget -qO- "https://kolm.ai/v1/registry/list?page=1&per=25&sort=trending"

Every response ships an X-Kolm-Receipt header so the response itself is replayable. See K-score methodology for what each score means.
