How it works

A rule-class classifier for the kind of triage that PR review bots want to automate: read a review comment, decide whether it is a security flag, a performance note, a style nit, a missing test, a docs gap, or a refactor request. The artifact is small enough to run inline on every PR comment without a model server in the loop.
