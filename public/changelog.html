v11.7

Frontier-serving engines and a 4-step train wizard.

2026-05-15
  • 4 frontier serving-engine backends: vLLM, SGLang, TGI, TensorRT-LLM. All OpenAI-compatible, env-var-detected, with realistic 7B-INT8 throughput baselines (2200 / 2500 / 1800 / 3000 tok/s). Backend registry grew from 14 to 18.
  • /train: in-account 4-step wizard. Pick a base, upload a recipe, set a gate, watch a simulated 12-step compile log.
  • kolm diff: git-style CLI diff between two .kolm files. Compares CID, base, K-score, recipe, compliance pack, version. Has --json for CI.
  • /verify-prod polish: mobile breakpoint at 560px, new "Works on your platform" section (desktop, mobile, offline browser).
  • Routing: 7 new /vs-* redirects consolidate marketing surfaces under /compare/*. 8 wave-5 pages got nav normalization.
v11.6

14 backend adapters and post-signup UX.

2026-05-14
  • 14 backend adapter layer: Python runtime mirror of the JS compute layer. 6 local (CPU, CUDA, MPS, MLX, ROCm, DirectML), 8 remote (Modal, RunPod, Together, Vast, Lambda, Replicate, fal, remote-SSH). Realistic 2026 pricing in every quote().
  • /onboarding: 5-step post-signup wizard. Key copy, install tabs, starter templates, run + verify, next steps. State persists in localStorage.
  • /dashboard welcome banner: shows on first sign-in when compiles_count is 0, dismisses to localStorage, CTA points at /onboarding.
v11.5

Browser verifier, underwriter angle, compliance hub.

2026-05-13
  • /verify-prod: browser-side .kolm verifier. Drop a file on the page; six checks run client-side via crypto.subtle.digest. Nothing uploaded.
  • /insure: receipts framed as underwriting evidence. Three loss scenarios (E&O, professional liability, cyber). underwriting@kolm.ai intake.
  • /compliance hub: 6 framework cards (NIST AI RMF, HIPAA, SR 11-7 live; ISO 42001, EU AI Act, SOC 2 with ETAs).
  • /compliance/nist-ai-rmf: 31-subcategory mapping onto manifest + receipt + CLI verbs against the July 2024 GenAI profile.
  • 3 case studies: healthcare PHI redactor (K=0.982, 4M transcripts/qtr), finance SR 11-7 (OCC exam, 9-day evidence package), legal contract extraction (AmLaw 200, 14k contracts).
  • Distribution: browser-extension (MV3), Homebrew formula, winget manifest, apt control. Closes the "available on Homebrew?" infra-scorecard checkbox.

  • Golden Path tutorials: /tutorials/phi-redactor, /tutorials/openai-drop-in, /tutorials/ci-verify. TechArticle + BreadcrumbList JSON-LD.
  • Trust + procurement triad: /soc2 (Type-1 window + Type-2 letter dates), /sbom (CycloneDX 1.5 weekly), /bounty (8-class payout table).
  • /baa: DocuSign-via-mailto request form. 7 fields, no backend.
  • /integrations/cloudflare and /integrations/github-actions: reference workflows.
  • scripts/audit-buyer-journey.sh: 30-probe weekly canary across 6 funnel stages. Catches surface drift before users do.
