path 03 · compile a custom .kolm · ~60 seconds

Compile your own from your examples.

The CLI posts your task and examples to the configured cloud, gates on K-score, and downloads the signed artifact to ~/.kolm/artifacts. The .kolm runs anywhere you put it.

01 Install the CLI 
# macOS / Linux / Windows · needs Node 20+
npm i -g github:sneaky-hippo/kolmogorov-stack
kolm version
02 Sign up (one call, free) 
# provisions a tenant + API key, saved to ~/.kolm/config.json
kolm signup --email you@example.com

# or, if you already have a key from /signup
kolm login --key ks_...
03 Drop in your examples 
# newline-delimited JSON · {input, output} pairs · keep it small to start
echo '{"input":"checkout fails on Android","output":"bug"}' > tickets.jsonl
echo '{"input":"please add export to CSV","output":"feature"}' >> tickets.jsonl
echo '{"input":"where do I reset MFA","output":"support"}' >> tickets.jsonl
04 Compile 
kolm compile "classify support tickets" --examples ./tickets.jsonl
ok wrote ./tickets.kolm  · K=0.91  · receipt rcp_8f2a...
05 Run locally 
kolm run ./tickets.kolm '{"input":"checkout fails on Android"}'
→ bug  · latency 82us  · signed receipt
