LangChain JS

LangChain JS speaks OpenAI and Anthropic over their published HTTP APIs. Pass configuration.baseURL (OpenAI) or clientOptions.baseURL (Anthropic) to point the chain at the local kolm proxy. Every call into the chain becomes a captured (input, output) pair you can later distill.

Install

npm i langchain @langchain/openai @langchain/anthropic
npm i -g kolm

Start the proxy

kolm capture --provider openai --as triage-chain --namespace triage
# proxy listening on http://127.0.0.1:8787

ChatOpenAI

import { ChatOpenAI } from "@langchain/openai";

const model = new ChatOpenAI({
  model: "gpt-4o-mini",
  apiKey: process.env.OPENAI_API_KEY,
  configuration: {
    baseURL: "http://127.0.0.1:8787/v1",
    defaultHeaders: {
      "x-upstream-api-key": process.env.OPENAI_API_KEY,
      "x-kolm-namespace": "triage",
    },
  },
});

const result = await model.invoke("classify: order delayed");
console.log(result.content);

ChatAnthropic

import { ChatAnthropic } from "@langchain/anthropic";

const model = new ChatAnthropic({
  model: "claude-3-5-sonnet-latest",
  apiKey: process.env.ANTHROPIC_API_KEY,
  clientOptions: {
    baseURL: "http://127.0.0.1:8787",
    defaultHeaders: {
      "x-upstream-api-key": process.env.ANTHROPIC_API_KEY,
      "x-kolm-namespace": "triage",
    },
  },
});

Tools and structured output

LangChain's bindTools, withStructuredOutput, and streaming all pass through unchanged. The proxy forwards the upstream response body byte-for-byte; only the metadata sidecar (latency, namespace, token counts) is appended to the local lake.

Verify the proxy sees traffic

kolm capture status --namespace triage
# pairs captured:    38 / 1000
# pairs until distill: 962

kolm distill --namespace triage  # once the threshold trips

Env-var shortcut

If you do not want to touch the LangChain config, set the env vars before importing:

// before any import:
process.env.OPENAI_BASE_URL = "http://127.0.0.1:8787/v1";
process.env.ANTHROPIC_BASE_URL = "http://127.0.0.1:8787";

The official OpenAI and Anthropic SDKs (which LangChain wraps) honor these env vars at construction time. Setting them before require or import avoids the config block entirely.
