kolm CLI reference

The kolm CLI ships every verb you need to capture prompts, compile them into a smaller artifact, verify the receipt chain, and ship the result. Run kolm --help for the full one-screen list; this page links to the long-form pages for the verbs that need them.

Install

npm i -g kolm
kolm --version
kolm --help

One package, no native build step. On first run the CLI writes ~/.kolm/config.json with the default base URL and creates the local state directory.

Verbs, grouped

Build an artifact

kolm seedsnew | generate | list | bootstrap. Scaffold and manage seeds.jsonl training corpora. kolm nlNatural-language recipe scaffolder. Plain English in, spec.json plus seeds out. kolm modelslist | info | recommend | pin | devices. Inspect the frontier catalog and pre-cache weights. kolm quantizeint4 | int8 | gptq | awq. Local worker that quantizes a checkpoint.

Capture and distill

kolm keyslist | rotate | fingerprint | export. Ed25519 signing-key lifecycle. kolm distillAuto-distill a captured namespace into a local LoRA via the trainer bridge.

Dev-agent helpers

kolm installWire the kolm proxy into Codex, Claude Code, Cursor, Windsurf, Aider, Gemini CLI. kolm connectDrop-in OpenAI / Anthropic / Ollama / LiteLLM / vLLM / SGLang / TGI connectors.

Full verb list

Run kolm --help for the one-screen list. Every verb supports --help for its own subcommands and flags.

kolm init                 scaffold kolm.yaml + .kolm/ at cwd
kolm build <name>         one-shot new + seeds + compile + verify
kolm compile "<task>"     cloud-compile a task into a .kolm artifact
kolm run <art.kolm>       execute a .kolm against an input
kolm verify <art.kolm>    full verification + optional printable binder
kolm inspect <art.kolm>   manifest + recipes + signature
kolm list                 show every local .kolm artifact
kolm capture --provider   drop-in proxy that captures (input, output) pairs
kolm distill --namespace  auto-distill a namespace into a local LoRA
kolm seeds new            scaffold seeds.jsonl from a template or brief
kolm nl "<request>"       natural-language recipe scaffolder
kolm models list          inspect the frontier model catalog
kolm quantize --local-worker  quantize a checkpoint via the local worker
kolm keys rotate          Ed25519 signing-key rotation
kolm doctor               sanity-check env (config, cloud, docker, project)
kolm whoami               echo current tenant + plan + base
kolm version              print version (CLI + server contract)

Config and environment

VariableEffect
KOLM_BASECloud endpoint. Default https://kolm.ai.
KOLM_API_KEYBearer token. Overrides ~/.kolm/config.json.
KOLM_DEBUGAny non-empty value prints stack traces on errors.
KOLM_NO_REST_HINTSuppress the REST equivalent hint after run / verify.
