kolm distill

Auto-distill a captured namespace into a local LoRA via the kolm trainer bridge. Captures from kolm capture cross a threshold, the trainer runs against a chosen base model, the result ships as a signed .kolm artifact.

Usage

kolm distill --namespace <n> [--base-model <name>] [--target <size>]

kolm distill --local-worker --spec <file> --seeds <file> --out <dir>
                            [--mode stub|collect|full|doctor]
                            [--teacher <vendor:model>]
                            [--student-base <name>]
                            [--distillation-method lora|qlora|full-ft|prompt-distill]
                            [--split-seed N] [--redact phi|pci|multi|none|auto]

Defaults

FlagDefault
--base-modelQwen/Qwen2.5-3B-Instruct
--targetphi-3-mini
--redactauto (the worker detects redact_class from the spec)
--distillation-methodDerived: lora when --mode=full with Python ML stack present, prompt-distill otherwise.

End-to-end flow

Step 1

Capture pairs against a namespace

Point your existing app at the local proxy and let it accumulate (input, output) pairs against a named namespace.

kolm capture --provider openai --as ticket-classifier --namespace tickets
# proxy listening on http://127.0.0.1:8787
# point OPENAI_BASE_URL there in the app
Step 2

Wait for the threshold

Default distill threshold is 1000 captured pairs per namespace. Track progress.

kolm capture status --namespace tickets
# pairs captured:    438 / 1000
# pairs until distill: 562
Step 3

Distill the namespace

Once the threshold trips, run the distill. The trainer chooses a recipe (LoRA when full ML stack present, prompt-distill otherwise) and emits a signed artifact.

kolm distill --namespace tickets
# base-model:    Qwen/Qwen2.5-3B-Instruct
# target:        phi-3-mini
# method:        lora  (trainer bridge ok)
# K-score:       0.91  (pass; gate 0.88)
# artifact:      ~/.kolm/artifacts/tickets-2026-05-18.kolm
Step 4

Verify and ship

Re-check the receipt chain and inspect the artifact before promoting.

kolm verify ~/.kolm/artifacts/tickets-2026-05-18.kolm
kolm inspect ~/.kolm/artifacts/tickets-2026-05-18.kolm
kolm run ~/.kolm/artifacts/tickets-2026-05-18.kolm 'classify: order delayed'

Redaction classes

The trainer accepts --redact to mask sensitive substrings before any teacher call. Classes: phi (HIPAA Safe Harbor identifiers), pci (card data), multi (both), none (off), and auto (read redact_class from the spec). --no-redact is equivalent to --redact=none and is rejected when --redact=<class> is also passed.

Honest scope

The distill verb runs against a captured namespace or a local spec plus seeds. The hosted compile path is a separate verb (kolm compile). Local distill needs a working Python ML stack for full LoRA mode; --mode doctor probes the worker and reports what is missing.
