What the air-gap actually gets you

No cloud round-trip, no shared GPUs, no token telemetry. The compile server runs inside your enclave; the .kolm artifact lives in your repo; the receipt chain is verifiable offline.

IL5/IL6 enclaves

The self-hosted compile server kit ships as a Docker image and Helm chart. Deploy into your IL5 or IL6 enclave, point your build pipeline at it, and never reach back across the diode.

ITAR-bounded data

Training corpus, captured prompts, and the resulting weights all stay inside the export-control boundary you already maintain. No third-party API calls in the build path.

Signed, reproducible artifacts

Every .kolm carries a receipt chain (spec, seeds, split, train, evals, signatures) that re-verifies offline. A different machine on the air-gap side computes the same artifact_hash bit-for-bit.
